## Common Workflows

### Create & Verify Company
```
1. Companies → Create Company
2. Fill required fields
3. Submit → Auto-verification starts
4. Wait for completion (< 2 hours)
5. Review results
```

### Review High-Risk Company
```
1. Filter by High Risk (71-100)
2. Open company detail
3. Review discrepancies
4. Check data sources
5. Mark as Reviewed/Flagged
6. Add notes
7. Export report if needed
```

### Correct Data
```
1. Open company detail
2. Click "Correct Data" on field
3. Enter new value
4. Provide reason
5. Submit → Pending admin approval
```

### Export Report
```
1. Open company detail
2. Click "Export Report"
3. Select format (PDF/CSV/JSON/HTML)
4. Download file
```
